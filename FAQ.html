Purpose

The purpose of this website is to show a few examples of how do find influental doctors in a specific area; in this example, psychiatrists in the Philadelphia area.

Directions

Results table (Home page)
The table provides a list of all doctors in the area that can be exported to other formats.

  • clicking a column sorts the column
  • search will filter the table based upon your input
  • navigation allows you to go to a specific page
  • the export buttons allow you to download the data in varous formats.

Network Diagram
The network diagram is visual of the relationships between the doctors. This type of diagram can provide insight in how the doctors communicate with their peers. This diagram is also interactive; some of the features are:

  • The top 10 most inluential doctors are bolded.
  • The colors and the bubbles respresent the natural groupings of the doctors. The groupings are based off of communications in the network.
  • Each node can be draggedd to a location, in order to provide better visability.
  • Hoovering over a node displays that doctor's name.
  • Clicking on a node provides demographic data on that doctor
  • Double clicking a node will highlight that doctors nearest neighbors.
  • Clicking a node size option adjust the size of the node in propotion to the parameter selected. i.e selecting prescriptions changes node size to represent the number of prescriotions they wrote in a year.

3D Network Diagram

  • left click rotates the diagram
  • mouse wheel will zoom in and out
  • right click pans the diagram
  • Hoovering over a node displays that doctor's name.
  • Clicking on a node provides demographic data on that doctor
  • Clicking a node size option adjust the size of the node in propotion to the parameter selected. i.e selecting prescriptions changes node size to represent the number of prescriotions they wrote in a year.

FAQ

  • Is the data presented real data? Yes, if you needed a sales list of Philadelphia psychiatrists the data here could be used as your target list.
  • Where did the data come from? Bluejacket gathers data daily from numerous sources such as the Centers of Medicare and Medicaid Services, National Provider Identifier Registry, Census Bureau, Clinical Trials, Scientic Journals, social media and news casts then combines this data into a single coherent story using AI processes.
  • Can other target lists be generated? Yes, our database has billions of records and contains every healthcare provider in the United States along with every medical journal article and every drug produced which allows us to filter the data in thousands of ways for unparalleled accuracy.
  • How many doctors can be in a network? An unlimited amount; however, the visual displays get very busy making it difficult to gain insight from the diagrams.
  • How do you determine influence? The alogrithms used are based off network science, social science and applied mathematics. The process is similar to how Google finds the best (most influential) page based upon your search criteria. The home page table field labeled "Rank" is a rank of the doctor's influence.
  • Can you prove you are measuring influence? Yes, we have run several research studies that have statistically proven we are measuring influence.
